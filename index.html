Intro

Problems aren't so complex if you stay a second to think about them. To solve an issue you must think in a calm way and leave aside the stress which is a productivity and solution solver killer.

Throughout my lifetime I've always appreciated the people who put effort in their work. They evolve as persons alongside their skillset. You can always find benefits when a good work is done.

Since my highschool years I've always struggled with having everything I work on to be clearly understood and the result to be perfect. This is a double-edged sword: as a developer you need to dive into many technology stacks so that you can achieve it and sometimes it has its bad and goods: the knowledge earned in exchange for time and focus spent. But doing so, it helps to be aware of the time and resources spent on specific technologies.

In this domain you need to have to have faith in your teammates/colleagues. You need to know how and when you should delegate some tasks to them, describe the work that needs to be done and help them when they have impediments.

Bachelor thesis FestiMove project 2017

FestiMove is a real time application in android that uses a secure spring boot application as a backend server.

The application was designed to show in real time the whereabouts of the users. The communication between the client and the server was STOMP over websocket based and https for registration/authentication. The server security was enhanced by using OAuth 2.0.

The backend server used 4 spring major components.(Security, OAuth 2.0, Websocket, Data JPA).

The android application was implemented having in mind the dependency injection architecture. The method used was pure dependency injection. The app also integrated google’s APIs like Fused Location Provider/Google Maps.
